# 🤖 AI Code Review Assistant

An AI-powered code review application built with Next.js, FastAPI, and OpenRouter.

## 🚀 Features

- AI-powered code review
- Monaco code editor
- Multiple programming languages
- Markdown review output
- FastAPI backend
- Next.js frontend

## 🛠️ Tech Stack

### Frontend

- Next.js
- React
- Tailwind CSS
- Monaco Editor

### Backend

- FastAPI
- Python
- OpenRouter API

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/aloksuyal0/ai-code-review-assistant.git
```

### Backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
